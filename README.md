This project demonstrates how to install, configure, and host a **static HTML website** using the **Nginx web server** on **Lubuntu (Linux)**.
## Project Overview

The goal of this project is to understand how web servers host and serve content locally.
Nginx was used to deploy a static HTML website and verify its accessibility through `http://localhost`.

---

## Steps Performed

1. Installed and configured Nginx on Lubuntu.
   - It involved a series of commands to configure nginx server.
3. Placed the website files inside `/var/www/html/`.
4. Replaced the default Nginx page with a custom HTML file.
5. Restarted the Nginx service and verified the deployment.

---

##  Tools & Technologies

* **Operating System:** Lubuntu (Linux)
* **Web Server:** Nginx
* **Frontend:** HTML
